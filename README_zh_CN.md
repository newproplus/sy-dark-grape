## 主要特点

* 尽量使用思源笔记的默认样式，只修改必要的样式。

* 编辑器内不修改用户文本（普通文字）的样式，只有项目符号 / 表头和代码块 / 链接 / 引用等的颜色和背景色等样式和默认样式不同。

* 用颜色明确界面的各个元素，增加渐变和半透明效果，避免深色模式下的视觉疲劳。
  
* 主要的部分用各种紫色， 点缀的部分用绿色。

## 推荐的字体

"Libration Mono": 对于“I L i l 1”有明显的区别，并且粗体和符号显示得很好。
