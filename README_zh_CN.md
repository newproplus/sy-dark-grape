基于 [sy-dark-purple](https://github.com/frostime/sy-dark-purple) 0.4.10， 并大幅度修改了配色， 特点如下：

* 不修改编辑器内用户文本的样式
  
* 编辑器内只有列表的项目符号的字体颜色 / 表头背景色等极少数样式和默认样式不同。

* 用颜色明确界面的各个元素， 避免深色模式下的视觉疲劳。
  
* 只包含 css 样式，无风险
